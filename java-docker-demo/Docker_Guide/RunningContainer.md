#
# Running the first Docker Container 

```markdown
- Pull the image from the Docker hub and Run the Conatiner 
# Terminal Commands 
- docker --version 
- docker ps -a
- docker images 

# Docker Help 
    - docker help 

# Pull the Image from docker hub
    - docker pull "image name"
    - docker run hello-world

# Search the Image from docker Hub
    - docker search hello-world
    - docker pull hello-world

# Build the container  Image 
    - docker create hello-world
    - docker create myapp1

# Running the Conatiner/image
    - docker run "containerID"

# Removing the running Images from the docker 
 - 1. stop the container
    -  docker stop "Cid"
    - docker pause ""Cid
 - 2. docker rmi "containerID"


```
